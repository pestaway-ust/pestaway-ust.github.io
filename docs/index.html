Pestaway: A Pest Management Chatbot with Integrated Speech Capability

A bilingual conversational assistant that integrates speech input/output to help Filipino farmers identify pests, learn safe mitigation approaches, and access pest management information in English and Tagalog.

üìÑ Abstract

We present Pestaway, a bilingual conversational assistant for Filipino farmers that uses Large Language Models to provide interactive and speech-enabled pest management solutions. Addressing challenges farmers faced with limited access to accurate pest control information, Pestaway integrates Whisper speech-to-text and Google Text-to-Speech technologies with a fine-tuned SeaLLM model, enabling natural voice-based and text-based interaction in both English and Tagalog. The system supports both voice and text modalities for flexible interaction during fieldwork. We conclude with performance benchmarks, user satisfaction analysis from farmers and agricultural students, and recommendations for future enhancements. The source code can be accessed via https://github.com/pestaway-ust/pestaway.

üó∫Ô∏è System Overview

Pestaway is a conversational assistant that integrates speech input/output to help users quickly identify household and agricultural pests, learn safe mitigation approaches, and log actions. The system employs a three-component pipeline: users interact through Whisper STT for speech recognition, the fine-tuned SeaLLM-v3-1.5B-Chat model processes the request and generates pest management guidance, and Google TTS (gTTS) delivers the response through natural speech output. The system is designed for modest hardware and field deployment.

üíª User Interface

Pestaway offers an interactive platform with options to converse via text or speech input, providing flexible pest management assistance for Filipino farmers.

üìä Results

The evaluation compared different training methods: Supervised Fine-Tuning (SFT), SFT with Direct Preference Optimization (DPO), and Odds Ratio Preference Optimization (ORPO). Metrics include METEOR (translation quality), BERTScore (semantic similarity), and Average Perplexity (model confidence).
